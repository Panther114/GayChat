Desktop onboarding

Set up Gchat once, then sign in normally.

The desktop app opens the same hosted Gchat service and adds a few Windows-only conveniences.

Same hosted app

You will use the official Railway deployment, not a separate local server.

Private group keys

Encryption keys stay on your device. The server does not store them.

Desktop conveniences

Gchat adds tray behavior, native notifications, and optional launch at startup.

Automatic updates

New desktop releases download in the background and install after restart approval.

Connection check

Confirm that the official hosted server is reachable.

This build always connects to the same Railway deployment so setup stays simple and consistent.
