Desktop onboarding

Set up Gchat once, then sign in like the web app.

This desktop app wraps the same Gchat interface you already have on the web, then adds Windows features like tray behavior, launch-at-startup, native notifications, and automatic GitHub-based updates.

End-to-end encrypted groups

Messages are encrypted in the client before they are sent. Group keys stay on your device and are not stored or transmitted by the server.

Hosted online service

This desktop build is locked to the Railway deployment and requires an internet connection. When connection quality is poor, the app keeps showing status instead of entering an offline mode.

Account security

User passwords are hashed on the server, sessions are signed, and notifications plus startup behavior stay inside the local desktop shell.

Windows-native behavior

You get tray controls, unread badge overlays, native notifications, and automatic update downloads from GitHub releases.

Connection check

Your desktop app always connects to the official hosted server.

The server URL is locked for this build so setup stays simple and support stays predictable.
